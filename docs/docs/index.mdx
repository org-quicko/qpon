# Introduction

Qpon is a self-hosted coupon management system designed for businesses to create, manage and track coupons easily. The API-first design ensures seamless integration with your existing systems, making it easy to automate workflows and manage coupon data in real time.

## Features

Features:

* <b>Self-Hosted</b><br/>
     Deploy Qpon on your own servers for full control over your data and security.
* <b>RESTful API</b><br/>
     Integrate Qpon with your existing systems using its REST API, allowing automated coupon management, real-time tracking, and reporting.
* <b>Multi-tenant</b><br/>
     Manage multiple brands, stores, or campaigns from a single platform with dedicated API endpoints for each.
* <b>Custom Coupon Rules</b><br/>
     Define custom rules for coupon creation, including value-based discounts, usage limits, or specific redemption conditions via API.
* <b>Real-Time Tracking</b><br/>
     Monitor coupon usage, redemption rates, and campaign performance in real-time.
* <b>Role-Based Access Control</b><br/>
     Define user roles and permissions for API access, ensuring secure and controlled access to coupon data.
